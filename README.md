# 技术复习文档

本仓库包含 Android、Flutter 和 HarmonyOS 技术的综合文档，专为面试准备和技术参考而设计。

## 目录结构

- **android_native_tech_review/**: Android 原生开发文档
- **flutter_tech_review/**: Flutter 框架文档
- **harmony_next_tech_review/**: HarmonyOS Next 开发文档

## 内容概述

### Android 原生
- 核心组件（Activity、Service、BroadcastReceiver、ContentProvider）
- UI 系统（XML 布局、Jetpack Compose、布局管理器）
- 数据存储（SharedPreferences、文件存储、SQLite）
- 网络编程（OkHttp、Retrofit）

### Flutter
- 核心概念（Dart 语言、Flutter 架构、Widget 系统）
- 状态管理（Provider、Bloc、GetX）
- 路由和导航
- 网络请求和数据处理
- 本地存储解决方案
- 性能优化
- 原生平台交互
- 常用第三方库

### HarmonyOS Next
- 核心概念（ArkTS 语言、项目结构、开发环境）
- 组件和布局设计
- 状态管理机制
- 路由和导航管理

## 使用方法

本文档旨在作为技术面试、学习指南和开发最佳实践的参考资料。每个部分都包含核心概念、实现原理、使用场景、常见问题及解决方案、代码示例和面试问题。
