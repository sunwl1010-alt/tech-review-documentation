# Tech Review Documentation

This repository contains comprehensive documentation for Android, Flutter, and HarmonyOS technologies, designed for interview preparation and technical reference.

## Directory Structure

- **android_native_tech_review/**: Android native development documentation
- **flutter_tech_review/**: Flutter framework documentation
- **harmony_next_tech_review/**: HarmonyOS Next development documentation

## Content Overview

### Android Native
- Core components (Activity, Service, BroadcastReceiver, ContentProvider)
- UI system (XML layout, Jetpack Compose, layout managers)
- Data storage (SharedPreferences, file storage, SQLite)
- Network programming (OkHttp, Retrofit)

### Flutter
- Core concepts (Dart language, Flutter architecture, widget system)
- State management (Provider, Bloc, GetX)
- Routing and navigation
- Network requests and data processing
- Local storage solutions
- Performance optimization
- Native platform interaction
- Common third-party libraries

### HarmonyOS Next
- Core concepts (ArkTS language, project structure, development environment)
- Components and layout design
- State management mechanisms
- Routing and navigation management

## Usage

This documentation is intended to be used as a reference for technical interviews, study guides, and development best practices. Each section includes core concepts, implementation principles, usage scenarios, common issues and solutions, code examples, and interview questions.
